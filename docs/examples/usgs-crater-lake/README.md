# USGS Crater Lake Fact Sheet

This is the recommended compact PDF example. It is short, visual, and produces a small ZIP with Markdown plus extracted images.

## Files

- [source.md](source.md): source URL, rights notes, and input metadata.
- [output-preview.md](output-preview.md): short excerpt from the converted Markdown.
- [zip-contents.txt](zip-contents.txt): generated ZIP file listing.
- [images/](images): two small extracted JPEG files used by the rendered preview.

## Conversion Summary

- Input type: PDF
- Source size: 298,655 bytes
- PDF pages: 2
- Converted Markdown: 9,389 bytes
- Result ZIP: 212,953 bytes
- Extracted images: 5 JPEG files
- Runtime metadata: H100, `marker-pdf`, `pdftext_workers=4`

The output has clear headings, paragraphs, image references, and figure captions. This is the best first example for showing image extraction without a large source document.
