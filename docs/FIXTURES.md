## Modal PDF Fixture

`modal-converter/tests/part.pdf` is a one-page synthetic PDF generated for optional Modal/orchestrator tests. It contains only fixture text written for this repository and is safe to redistribute with the project.

## Intended Use

These fixtures are small and intentionally simple. They verify upload flows, file-type handling, and test plumbing. They are not representative benchmark documents for conversion quality.
