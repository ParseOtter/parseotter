# Public Assets

These assets were generated from the local ParseOtter production build using synthetic EPUB fixtures and mocked local API responses.

- `architecture.svg`: architecture diagram used by the top-level README.
- `parseotter-upload-screen.png`: upload surface with synthetic EPUB files selected.
- `parseotter-markdown-preview.png`: Markdown preview dialog with a synthetic ZIP result.
- `parseotter-flow.webm`: short UI recording covering a processing row, upload selection, preview, and download controls.

No private or user-provided documents are shown in these assets.
