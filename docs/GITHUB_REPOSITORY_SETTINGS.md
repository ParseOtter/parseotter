## Features

- Issues: enabled.
- Pull requests: enabled.
- Discussions: disabled unless maintainers want to run community discussion in GitHub.
- Wiki: disabled unless maintainers want to maintain separate docs.
- Projects: optional.

## Security

- Enable private vulnerability reporting if available.
- Confirm `SECURITY.md` has the final maintainer contact path.
- Confirm branch protection requires CI and security scan before merge.
- Keep GitHub Actions enabled for CI and security workflows.
